# Animated Side Navigation Menu

Uma barra de navegação lateral (Side Menu) moderna e animada desenvolvida exclusivamente com HTML5 e CSS3. O projeto apresenta um menu retrátil com transições suaves, ideal para otimizar o espaço de tela em painéis de controle, dashboards e aplicações web.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica dos links, ícones e contêineres de navegação.
* CSS3: Estilização completa, uso de variáveis CSS, gerenciamento de estados e animações de transição (`transition`).
* Flexbox: Alinhamento vertical e horizontal dos itens internos e dos ícones de redes sociais.

## Funcionalidades e Técnicas Aplicadas

* Animações com CSS Puro: Transições fluidas aplicadas à abertura, fechamento e interações de *hover* nos links, sem a necessidade de JavaScript.
* Menu Lateral Estruturado: Organização de links essenciais, incluindo Home, Clients, Services, Settings, Features, About us, Languages e Contact us.
* Cabeçalho de Controle: Seção superior com o título "Side Menu" e um botão de fechamento (X) integrado nativamente.
* Rodapé com Ícones Sociais: Barra inferior discreta com links e ícones para plataformas como Facebook, Twitter, GitHub e YouTube.
* Design de Alto Contraste: Layout construído em um esquema de cores escuro (*Dark Mode*) contra um painel principal claro para excelente legibilidade.
